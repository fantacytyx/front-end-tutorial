# Front-end-tutorial

[Web前端基础学习路线]（https://wuzhengyan2015.github.io/front-end-tutorial/)

## 路线图的修改
路线图用[Balsamiq](https://balsamiq.com/products/mockups/)绘制. 文件在 `/img/*.bmpr` . 需要修改的话, 打开 Balsamiq, 点击 **Project > Open Project**.	

## 运行
采用vuepress构建

yarn install

yarn dev // 开发

yarn deploy //部署
